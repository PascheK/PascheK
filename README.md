## 🚀 Compétences
- **Frontend** : Vue.js, NextJs, HTML, CSS, JavaScript, TypeScript
- **Backend** : PHP, Spring Boot
- **Base de données** : MySQL
- **Autres** : Git, Docker, CI/CD, API REST

## 🔥 Projets
